Hypertext markup Language(HTML) is the standard markup language for creating web pages and web applications

Cascading Style sheets(CSS) is the style sheet language used for describing the presentation of a document written in a markup language like HTML

How to create this page
  1. Link a stylesheet.
  2. Give title to the HTML page.
  3. Create a header
  4. Draw a horizontal line.
  5. Write text inside Paragraph
  6. Hyperlink some text to redirect to https://www.w3schools.com/html/
  7. Write some text in Verdana font
  8. Insert an image with width=50 and height=60
  9. Write some text inside a div and style it
  10. Write some text inside a div and style it similar to the above div
  11. Insert a video using HTML5
  12. Create a list of tasks to perform to create the page
